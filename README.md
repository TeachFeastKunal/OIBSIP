## Getting Started

### Prerequisites

- Python 3.7+
- Jupyter Notebook or JupyterLab
- Required packages (listed in `requirements.txt`)

### Installation

1. Clone this repository:
```bash
git clone https://github.com/TeachFeastKunal/OIBSIP.git
cd OIBSIP
```

2. Install required dependencies:
```bash
pip install -r requirements.txt
```

3. Launch Jupyter Notebook:
```bash
jupyter notebook
```

## Projects & Tasks

### Task 1: Iris Flower Classification

**Objective:** Train a machine learning model to classify iris flower species based on measurements.

**Description:**
The iris flower dataset contains measurements of three species: **setosa**, **versicolor**, and **virginica**. Each species differs according to their measurements of sepal length, sepal width, petal length, and petal width. The task is to train a machine learning model that can learn from these measurements and classify iris flowers into their respective species.

**Key Skills:**
- Feature extraction and analysis
- Classification algorithms
- Model evaluation and validation

**Dataset:** Available in the Scikit-learn library or downloadable from external sources

[View Iris Flower Classification Notebook →](./Task_1_Iris_Flower_Classification/)

### Task 2: Unemployment Analysis with Python

**Objective:** Analyze unemployment trends and patterns during the Covid-19 pandemic.

**Description:**
Unemployment is measured by the unemployment rate, which is the number of unemployed people as a percentage of the total labour force. We have seen a sharp increase in the unemployment rate during Covid-19. Analyzing the unemployment rate is a valuable data science project to understand economic trends and impacts.

**Key Skills:**
- Time series analysis
- Data visualization
- Statistical analysis
- Trend analysis

### Task 3: Car Price Prediction with Machine Learning

**Objective:** Build a regression model to predict car prices based on various features.

### Task 4: Email Spam Detection with Machine Learning

**Objective:** Build an email spam detector using machine learning algorithms.

**Description:**
Spam mail, or junk mail, is a type of email sent to a massive number of users at one time, frequently containing:
- Cryptic messages
- Scams
- Phishing content

In this project, we use Python and machine learning to train a spam detector that can recognize and classify emails into **spam** and **non-spam** categories.

**Key Skills:**
- Text preprocessing
- Feature extraction (TF-IDF, Bag of Words)
- Classification algorithms
- Model evaluation

**Dataset:** Download from the provided source

[View Email Spam Detection Notebook →](./Task_4_Email_Spam_Detection/)

### Task 5: Sales Prediction using Python

**Objective:** Predict sales based on advertising spend and marketing strategies.

**Description:**
Sales prediction means predicting how much of a product people will buy based on factors such as:

Typically, product and service-based businesses use Data Scientists to predict future sales with every step they take to manipulate advertising costs. This project demonstrates how to build a sales prediction model using machine learning.

**Key Skills:**
